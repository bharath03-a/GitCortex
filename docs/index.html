Why GitCortex
Built for engineers who care about privacy

Every token of your source code stays on disk. No cloud round-trips, no LLM calls, no API bills — just a fast local graph that your AI assistant can query instantly.

🔒

100% Local & Private

KuzuDB embedded graph, fastembed local model weights. Source never touches a server. Works air-gapped.

⚡

Git-Hook Driven

Fires on post-commit, post-merge, post-rewrite, post-checkout. Incremental diffs only — hook path <100ms.

🌿

Branch-Aware

Each branch gets its own graph namespace. Switching branches swaps context. branch_diff shows what changed structurally between any two branches.

🤖

MCP-Native

Ships an MCP server (gcx serve) so Claude, Cursor, and any MCP-compatible assistant can query the graph with structured tools instead of grep.

Features
Everything your AI assistant needs to understand your code

From multi-language AST parsing to interactive 3D visualization — all running locally, all wired to your git workflow.

Multi-Language AST Parsing

tree-sitter backend — one API, every language. Structs, functions, methods, traits, calls, and imports extracted precisely.

Semantic Search

fastembed local embeddings — no API key, no rate limits. Ranked fuzzy search over names and qualified paths at MCP query time.

fastembed-rs BAAI/bge-small local weights

Clustering & God Node Detection

Deterministic label propagation over Contains+Calls edges. find_clusters surfaces module groupings. find_god_nodes finds high-in-degree bottleneck symbols.

LPA in-degree centrality deterministic

Interactive 3D Visualization

Axum-served React + Cosmograph force-graph. Color-coded by node kind, filterable, keyboard-navigable, WCAG-AA accessible. Exports to HTML, SVG, DOT, GraphML.

WebGL dark theme ARIA roles keyboard nav

Markdown Doc Ingestion

Headings become Section nodes. Code-span symbol mentions resolve to References edges cross-language — doc links tracked in the graph, not just source code.

pulldown-cmark NodeKind::Section EdgeKind::References

Incremental Indexing

Diffs from last indexed SHA, not HEAD~1. Idempotent — re-run safe. Only changed files parsed. Large monorepos stay fast.

Install
Three ways to get started

Pre-built binaries ship for macOS (arm64/x86_64) and Linux (x86_64/arm64). Windows users: use WSL2.

# Recommended — works on any Python 3.8+ env
pip install gitcortex

# Or with pipx for isolated install
pipx install gitcortex

# Or with uv
uv tool install gitcortex

# Wire up git hooks in your repo
gcx init

Wheels include pre-built binaries for macOS arm64/x86_64 and Linux x86_64/arm64. No Rust toolchain needed.

# Install from crates.io
cargo install gitcortex --locked

# Wire up git hooks
gcx init

# Start the MCP server
gcx serve

Requires Rust 1.82+. Builds KuzuDB from source — first build takes ~5 minutes.

# Install globally
npm install -g gitcortex

# Or with pnpm / bun
pnpm add -g gitcortex
bun add -g gitcortex

# Wire up git hooks
gcx init

npm package bundles the same pre-built binaries as the Python wheel.

# macOS arm64
curl -L https://github.com/bharath03-a/GitCortex/releases/latest/download/gcx-aarch64-apple-darwin.tar.gz | tar xz
sudo mv gcx /usr/local/bin/

# macOS x86_64
curl -L https://github.com/bharath03-a/GitCortex/releases/latest/download/gcx-x86_64-apple-darwin.tar.gz | tar xz
sudo mv gcx /usr/local/bin/

# Linux x86_64
curl -L https://github.com/bharath03-a/GitCortex/releases/latest/download/gcx-x86_64-unknown-linux-gnu.tar.gz | tar xz

MCP Server
Your AI assistant, now with a map

Add gcx serve to your .mcp.json. Claude, Cursor, and any MCP-compatible tool instantly gains structured access to your knowledge graph.

  • lookup_symbol(name)Find all definitions matching a name across the graph
  • find_callers(name)Who calls this function? Full call graph traversal
  • get_subgraph(name, depth)BFS neighborhood of a symbol — callers, callees, implementations
  • semantic_search(query)Ranked fuzzy search via local fastembed embeddings
  • find_god_nodes(min_in_degree)High-in-degree symbols — bottlenecks and hot modules
  • find_clusters(min_size)Label-propagation clusters over Contains+Calls graph
  • branch_diff(from, to)Structural diff between two branches
  • start_tour(seed?)Guided codebase tour — centrality-ranked or BFS from a seed

.mcp.json — drop in your project root

{
  "mcpServers": {
    "gitcortex": {
      "command": "gcx",
      "args": ["serve"],
      "env": {}
    }
  }
}

Works with Claude (Claude Code / claude.ai), Cursor, Windsurf, Continue.dev, and any other MCP host.
